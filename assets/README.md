# 🖼️ Assets

Pasta para armazenar imagens, diagramas, screenshots e outros recursos visuais utilizados nas anotações e projetos.

---

## 📁 Organização Sugerida

```
assets/
├── diagramas/        # Diagramas UML, fluxogramas, mapas mentais
├── screenshots/      # Capturas de tela de projetos e exercícios
└── README.md         # Este arquivo
```

---

## 📌 Convenção de Nomes

Use nomes descritivos e em minúsculas com hífens:

```
diagrama-uml-sistema-login.png
screenshot-app-mobile-tela-inicial.png
fluxograma-algoritmo-ordenacao.png
```

---

## 🔗 Como Referenciar nas Anotações

Use caminhos relativos nos arquivos Markdown:

```markdown
![Descrição da imagem](../../assets/screenshots/nome-do-arquivo.png)
```

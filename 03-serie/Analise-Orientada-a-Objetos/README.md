## 📋 Ementa Resumida

- Princípios da orientação a objetos (abstração, encapsulamento, herança, polimorfismo)
- Diagramas UML orientados a objetos (classes, sequência, colaboração)
- Padrões de projeto (design patterns — GoF)

- Utilizar UML para comunicar decisões de design.

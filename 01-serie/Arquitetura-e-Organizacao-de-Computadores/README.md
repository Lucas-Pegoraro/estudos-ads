## 📋 Ementa Resumida

- Representação de dados (binário, hexadecimal, ponto flutuante)
- Componentes de hardware: CPU, memória, barramento, E/S
- Arquitetura von Neumann e Harvard
- Pipeline e paralelismo
- Hierarquia de memória (cache, RAM, disco)

## 🎯 Objetivos

- Entender como o hardware executa instruções.
- Relacionar arquitetura de hardware com desempenho de software.
- Compreender a hierarquia de memória e seu impacto.
